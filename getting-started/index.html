Getting Started with Etherpad Plus¶

This guide will help you get started with Etherpad Plus in just a few minutes.

Accessing Etherpad Plus¶

  1. Open your web browser (Chrome, Firefox, Safari, or Edge)
  2. Navigate to your Etherpad Plus URL (provided by your administrator)
  3. Example: http://documents.yourdomain.com or http://localhost:11155
  4. You're ready to start! No login required.

Creating Your First Pad¶

Method 1: Create a New Pad¶

  1. Click "New Pad" in the top menu
  2. Enter a name for your pad (e.g., "Meeting Notes")
  3. Click "OK" or press Enter
  4. Start typing!

Method 2: Use the Default Pad¶

  • If you see a welcome pad, you can start editing immediately
  • Or create a new pad using the method above

Basic Editing¶

Typing and Editing¶

  • Type normally - Your changes appear in real-time
  • See other users - Their cursors and names appear in different colors
  • No save button needed - Everything saves automatically

Basic Formatting¶

  1. Select text by clicking and dragging
  2. Use the toolbar at the top:
  3. Bold (B) - Make text bold
  4. Italic (I) - Make text italic
  5. Underline (U) - Underline text
  6. Strikethrough (S) - Cross out text

Keyboard Shortcuts¶

  • Ctrl+B (Cmd+B on Mac) - Bold
  • Ctrl+I (Cmd+I on Mac) - Italic
  • Ctrl+U (Cmd+U on Mac) - Underline
  • Ctrl+Z (Cmd+Z on Mac) - Undo
  • Ctrl+Y (Cmd+Y on Mac) - Redo

Sharing Your Pad¶

Share the URL¶

  1. Copy the URL from your browser's address bar
  2. Send it to collaborators via email, chat, or any method
  3. They can open it and start editing immediately

Example URL Format¶

http://documents.yourdomain.com/p/YourPadName

Tips for First-Time Users¶

  1. Don't worry about breaking things - You can always undo (Ctrl+Z)
  2. Watch for colored cursors - These show where other users are editing
  3. Use headings - They help organize your document (see Plugin Features)
  4. Try the spell checker - It works automatically in your browser

Need Help?¶

  • Check the User Guide for detailed instructions
  • See Troubleshooting for common issues
  • Contact your system administrator for technical support
