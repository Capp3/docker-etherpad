What is Etherpad Plus?¶

Etherpad Plus is a self-hosted collaborative text editor that provides a Word-like writing experience. It combines:

  • Real-time collaboration - Multiple users can edit simultaneously
  • Word-like formatting - Familiar toolbar with headings, colors, images, and more
  • Document export - Export to HTML, ODT, or DOCX formats
  • Privacy-first - All data stays on your server
  • No accounts required - Start editing immediately
